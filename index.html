Connect Google Sheets
One-time 10-minute setup. Every round and session saves to your Sheet automatically.
1
Create your Google Sheet
Go to sheets.google.com → click "+" to create a blank sheet. Name it "Ian's Golf Tracker". Add 3 tabs: weekly_log, practice_log, profile.
2
Open Apps Script
In your Sheet: Extensions → Apps Script. Delete any default code. Paste the Apps Script code below (tap to copy).
TAP TO COPY APPS SCRIPT CODE
function doPost(e){...}
function doGet(e){...}
3
Deploy as web app
In Apps Script: Deploy → New deployment → Web app. Set "Execute as: Me" and "Who has access: Anyone". Click Deploy → copy the URL.
4
Paste your URL here
Once you have your deployment URL, paste it below and hit Connect.
